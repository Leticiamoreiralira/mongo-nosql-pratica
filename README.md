# MongoDB - Prática NoSQL

Este projeto demonstra a utilização do MongoDB Atlas para criação de banco de dados NoSQL.

## 🔗 Acesso ao banco (MongoDB Atlas)

Você pode visualizar os dados através do link:  
https://cloud.mongodb.com/v2/6407ae56cc09ef587e2ceded#/explorer/69cc573a4f8d3e1544343fd8/meu_banco/usuarios/find

---

## 📌 Etapas realizadas

- Criação de cluster no MongoDB Atlas  
- Criação de banco de dados (`meu_banco`)  
- Criação de coleção (`usuarios`)  
- Inserção de documento JSON  
- Consultas utilizando filtros  

---

## 📄 Exemplo de documento inserido

```json

  "ativo": true
```
